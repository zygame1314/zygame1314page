准备工作

在开始之前，请务必确保你已经：

  • 成功连接过 CCNU，并且用你的学号密码登录过一次。这是学校官方文档里唯一有用的信息，咱们得听话。

详细步骤 (Windows)

  1. 打开网络设置
    • 打开"控制面板" -> "网络和 Internet" -> "网络和共享中心"。
  2. 添加新网络
    • 在“网络和共享中心”里，找到“设置新的连接或网络”，点它！
    • 选择“手动连接到无线网络”，然后点“下一步”。
  3. 填写网络信息

    这里是关键，别填错啦：

    • 网络名：CCNU-AUTO
    • 安全类型：WPA2-企业 (或者 WPA3-企业，如果你的设备支持而且学校也开了的话)
    • 加密类型：AES

    填完点"下一步"。如果提示已存在同名网络，请先在 Wi-Fi 列表中找到 CCNU-AUTO 并“忘记”它，然后重新开始配置。

  4. 修改连接设置

    点击“更改连接设置”，进入更高级的配置。

  5. 配置安全选项卡
    • 网络身份验证方法：选择 Microsoft: 受保护的 EAP (PEAP)
    • 点击旁边的“设置”按钮，继续！
  6. 配置受保护的 EAP 属性

    这里是重中之重，千万别搞错！

    • 取消勾选 “验证服务器证书”。(不取消就连不上，问就是量子力学)
    • 选择身份验证方法：安全密码 (EAP-MSCHAP v2)
    • 取消勾选 “自动使用 Windows 登录名和密码(和域, 如果有)”。(不然你的 Windows 密码就泄露给校园网啦)
    • 勾选“启用快速重新连接”。(这个随意，勾上方便点)

    设置完点“确定”。

  7. 高级设置
    • 回到“安全”选项卡，点“高级设置”。
    • 勾选"指定身份验证模式", 下拉框选择"用户身份验证"。
    • 点击"保存凭据",输入用户名和密码：输入你在 CCNU 校园网认证时使用的账号和密码。
    • （可选）取消“启动成对主密钥(PMK)缓存”。
  8. 保存设置

    一路“确定”回去，保存所有设置。

  9. 连接 CCNU-AUTO

    在 Wi-Fi 列表里找到 CCNU-AUTO，点“连接”！如果弹出让你输入用户名密码，再输一遍你的 CCNU 账号密码。

见证奇迹的时刻！

如果一切顺利，你现在应该已经成功“复活” CCNU-AUTO 了！如果还是不行，可以尝试重启电脑，或者检查一下上面的步骤有没有遗漏。

小提示

  • 如果连不上，先检查账号密码有没有输错。
  • 确保“验证服务器证书”是取消勾选的状态。
  • “自动使用 Windows 登录名和密码”也一定要取消勾选！
  • 如果还是不行，可以尝试把上面步骤里的“可选”操作也做了。
  • 实在不行，咱还是老老实实用 CCNU 吧，毕竟活着最重要。
